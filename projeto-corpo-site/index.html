HTML, CSS e JavaScript

O corpo humano dos sites

HTML = O Esqueleto

O HTML é a estrutura do site, como os ossos e articulações do corpo humano.

<h1>Olá, mundo!</h1>
<p>Este é meu primeiro parágrafo.</p>

CSS = A Aparência

O CSS é a pele, roupas e estilo. Ele define cores, tamanhos, fontes e espaçamentos.

Caixa estilizada pelo CSS

JavaScript = O Movimento

O JavaScript é como os músculos e o cérebro: ele dá vida, ação e interatividade ao site.
